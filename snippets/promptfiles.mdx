## Introduction

Montelo Promptfiles help you keep your prompts separate from your code.

A promptfile ends with `.prompt` and contains an HTML-like syntax to define prompts.

For example, you can define a prompt like `weatherman.prompt`, and after running `npx montelo generate`, Montelo will
generate the code inside `prompts.ts` for you.

<CodeGroup>
```html prompts/weatherman.prompt
<System>
  You are a weatherman.
</System>

<User>
  {{message}}
</User>
```

```typescript prompts/prompts.ts
// Auto generated by Montelo. Run `npx montelo generate` to re-generate.
export const prompts = {
  weatherman: ({ message }: { message: string }) => [
    {
      role: "system",
      content: `You are a weatherman.`,
    },
    {
      role: "user",
      content: `${message}`,
    },
  ],
};
```
</CodeGroup>

<Note>Use `{{ }}` to pass in arguments.</Note>
