#### Description

This tool allows you to carry out Google searches using the **SerpAPI**.

#### How to use

1. Sign up and get a _free_ api key from [SerpAPI](https://serpapi.com/)
2. Set the `SERP_API_KEY` in your .env file
3. Import and use the tool in your code:

```typescript
import { Agent, Tools } from "@montelo/crews";

const someResearchAgent = new Agent({
  tools: [Tools.GoogleSearchTool],
  // other params...
});
```
