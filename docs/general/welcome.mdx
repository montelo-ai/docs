## What is Montelo?

Montelo makes it easy to generate synthetic data and fine-tune open-source LLMs.

With Montelo, you can:

- Create synthetic datasets from scratch or from samples of your existing data
- Fine-tune LLMs in one click
- Deploy an endpoint for inference

Montelo is available in a TypeScript and Python SDK.
