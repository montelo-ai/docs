#### Description

A tool designed to extract and read the content of a specified website. It is capable of handling various types of web pages by making HTTP requests and parsing the received HTML content. This is useful when you want to scrape:

1. **Product prices and descriptions**
2. **Posts and articles**
3. **Reviews and comments**
4. [Other cool use cases](https://medium.com/@RenderAnalytics/high-value-web-scraping-use-cases-86334ba24e5c)

#### How to use

Just import and use the tool in your code. All parameters are optional:

```typescript
import { Agent, Tools } from "montelo";

const WebScraperTool = Tools.WebScraperTool({
  // url: "amazon.com/...",
  // shouldFormatHtml: true,
  // divSelector: "div#content",
});

const someResearchAgent = new Agent({
  tools: [WebScraperTool],
  // other params...
});
```

#### Parameters

| Property           | Type      | Description                                                                                                                                                         | Required |
| ------------------ | --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------- |
| `url`              | _URL_     | If you want to limit it to only scrape a specific url                                                                                                               | No       |
| `divSelector`      | _String_  | If you want to scrape a specific section of the page                                                                                                                | No       |
| `shouldFormatHtml` | _Boolean_ | Formats the html into a more readable format. This is highly recommended as it reduces the number of tokens and improves the model's ability to extract information | No       |
