---
title: Multiple LLMs
description: How to use multiple LLMs with Montelo.
---

## Simple

Want to use OpenAI, Mistral, and more? Montelo supports multiple LLMs.

To use them, simply pass an object with the LLMs you want to use to the `Montelo` constructor.

```typescript
import { Montelo } from 'montelo';

const montelo = new Montelo({
  openai: {},
  mistral: {},
});

await montelo.openai.chat.completions.create();
await montelo.mistral.chat();
```

## With Trace

```typescript
import { Montelo } from 'montelo';

const montelo = new Montelo({
  openai: {},
  mistral: {},
});

const trace = montelo.trace();

await trace.openai.chat.completions.create();
await trace.mistral.chat();
```
