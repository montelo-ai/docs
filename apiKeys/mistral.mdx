---
title: Mistral
description: How API Keys work for Mistral.
---

The [Mistral Node.js SDK](https://github.com/mistralai/client-js) exposes a Javascript client to interact with the Mistral API.

The Mistral constructor takes in two options:

<ResponseField name="Mistral Constructor Options">
  <Expandable title="properties">
    <ParamField path="apiKey" type="string">
      Defaults to `process.env['MISTRAL_API_KEY']`.
    </ParamField>

    <ParamField path="endpoint" type="string">
      Override the default base URL for the API.
    </ParamField>
  </Expandable>
</ResponseField>

With Montelo, you still have access to these options, simply by adding them to the Montelo constructor.

```typescript
import { Montelo } from "montelo";

const montelo = new Montelo({
  mistral: {
    apiKey: "your-api-key",
    endpoint: "your-endpoint"
  },
});
```
